# Marathon Dashboards

This repository contains a collection of Grafana Dashboards that visualize data based on the Marathon Stats plugins in Aleph One.

## Structure

The dashboards are distributed in this structure:

```
Marathon Stats/
  Summary
  Marathon 1
  Marathon 1 Levels/
    1 - Arrival
    ...
  Marathon 2
  Marathon 2 Levels/
    ...
  Marathon Infinity
  Marathon Infinity Levels/
    ...
```

### Summary

This dashboard rolls up the stats from all games:
* Lists the games
